# Nomflix

- React, ES6를 공부하며 만든 영화 정보 사이트입니다.
- MovieAPI에서 데이터를 요청하여 사용하였습니다.
- 영화, TV의 인기 리스트, 방영 리스트 및 상세 페이지를 조회할 수 있습니다.
- Netlify에 배포하였습니다.
